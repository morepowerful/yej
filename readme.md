## Install
$ npm install yej -g

## Usage

- Generate a boilerplate.

```bash
$ yej init
```

- Install the dependencies.

```bash
$ cd yej-template-master(the project path)
$ yej install
```

- Start a local dev server.

```bash
$ yej server
```
> It will listen to the port 8080

- Build project.

```bash
$ yej build
```
